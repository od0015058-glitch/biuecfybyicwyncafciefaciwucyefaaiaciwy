# Primary Logo — Square Icon / Avatar

## Context for the AI image generator

Meowassist is a Persian/English Telegram AI assistant bot. It uses a cat (🐱)
as its mascot. The bot proxies LLM requests via OpenRouter and handles crypto
payments (Bitcoin, Ethereum, etc.) plus Iranian Rial card payments. The brand
identity is: **friendly, techy, Persian-influenced, cat-themed.**

- Domain: `meowassist.space`
- Telegram: `@Meowassist_Ai_bot`

---

## PROMPT — Primary Logo (1024×1024, square)

Minimal, modern logo mark of a stylized cat face merged with a circuit board
or neural network pattern. The cat has one eye shaped like a chat bubble and
the other like a glowing AI node. Color palette: deep purple (#6C3CE1) as
primary, electric cyan (#00E5FF) as accent, on a clean white or transparent
background. No text in the logo mark — just the icon. Flat design, no
gradients, no shadows. Suitable for use as a Telegram bot avatar (circular
crop friendly) and favicon. The cat silhouette should be simple enough to
recognize at small sizes but detailed enough to feel premium at full size.

## VARIATIONS (generate separately)

### Dark mode version
Same cat-circuit icon from the prompt above, placed on a dark navy (#1A1A2E)
background instead of white. The cyan accents should glow slightly against the
dark background. No text.

### Monochrome version
Same cat-circuit icon, rendered in a single color (deep purple #6C3CE1 on
transparent background). Suitable for watermarks, embossing, or single-color
printing.

### Inverted version
White cat-circuit icon on deep purple (#6C3CE1) solid background. For use on
dark-themed UI elements or as an app splash screen icon.
